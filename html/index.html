Program Information

This program will read in an input image, either in binary or ascii, and output it, again, in either binary or ascii, depending on command line arguments. If the number of arguments is 4, it will not perform any manipulation on the image; it will just output it as it was. If the number of arguments is 5, one of the arguments will specify what manipulation to put on the image. It will then output the manipulated image.

Compiling and Usage

Compiling Instructions:
None
Usage:
c:\> thpExam1.exe --outputtype basename image.ppm
c:\> thpExam1.exe [option] --outputtype basename image.ppm
d:\> c:\bin\thpExam1.exe --outputtype basename image.ppm
d:\> c:\bin\thpExam1.exe [option] --outputtype basename image.ppm

     --outputtype - type of data to output, either binary or ascii
     basename - name of output file
     image.ppm - name of input file
     [option] - type of manipulation on image
